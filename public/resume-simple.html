todoList/ 2017.04至今

项目功能简介 在线 todoList，用户添加并管理自己的待办事项，可以将事项标记为已完成、未完成状态，也可删除已 添加的事项。在第一版中，使用 localStore 对象将用户数据存储在本地，仅支持在本地用户不清除浏览器缓存时使用。第二版中， 利用 leanCloud 进行后台数据管理，用户可通过注册、登录后管理自己的账户，不再依赖本地浏览器缓存数据，一个浏览器终端也可由多个用户使用， 刷新页面或重启浏览器可保持本地用户的在线状态，其他功能尚在完善中。
涉及技术及收获 整个应用使用 React 框架搭建，使用 CSS 制作样式， 除了上述相关技术，以及基本的 JavaScript、HTML 外，在该项目开发的过程中还学习并实践了如何利用 webpack 对源码打包、 压缩，对 Promise 异步和 ES6 的新规范在项目中也进行了实践。

图片搜索/ 2017.3

项目功能简介 用户可通过输入框输入关键词（仅支持英文），点击搜索按钮后页面会展示搜索的返回结果，点击图片可跳转 到图片详情页，向下滚动到底后加载更多搜索结果，实现了懒加载效果
涉及技术及收获 项目使用原生 JavaScript 实现逻辑，结合 jQuery 对 DOM 操作，利用图片网站的开放 API 通过 JSONP 跨域请求数据，将返回结果加载到页面，并通过计算实现桶布局，另外利用 AJAX 技术使页面局部刷新实现懒加载，可让用户通过 滚动获取更多结果，提高用户体验
